.. _sec_physical:
Physical implementation
=======================

In order to use an SDRAM with this software the SDRAM must be physically connected to the xCORE in a certain way. 

The ports required are:

CLK	1 bit port
RAS	1 bit port
CAS	1 bit port
WE	1 bit port

DQ_AH	16 bit port


On the SDRAM

CLK
RAS
CAS
WE
DATA
...
